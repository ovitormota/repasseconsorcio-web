# Landing Page - Repasse Consórcio

Landing page informativa para a plataforma **Repasse Consórcio**, oferecendo detalhes sobre o funcionamento do sistema e captando leads interessados em leilões de consórcios de veículos.

## Tecnologias Utilizadas
- **Frontend**: Next.js (React)
- **Estilização**: Tailwind CSS
- **Markup**: HTML/CSS

## Configuração e Execução
Antes de iniciar, certifique-se de ter instalado:
- [Node.js](https://nodejs.org/)

### Passos para rodar o projeto (Next.js)
1. Instalar dependências:
   ```sh
   npm install
   ```
2. Iniciar o servidor de desenvolvimento:
   ```sh
   npm run dev  # Inicia o servidor Next.js
   ```
3. Acessar no navegador:
   ```
   http://localhost:3000
   ```
